# Alumni Network PWA - Mezunlar Derneği

## Vision
Mezunlar arasında güçlü bir profesyonel ve sosyal bağlantı ağı oluşturan, modern PWA platformu. Kullanıcılar mail + 8 haneli üyelik numarası ile giriş yapar, birbirleriyle mesajlaşır, etkinliklere katılır ve kariyer fırsatları yaratır.

**Target:** Türk üniversite/lise mezunları (çok yaşlı grup, çeşitli meslekler)

## Design
**Color Palette:**
- `--primary: 221 83% 53% (kurumsal mavi - #2563eb benzeri)`
- `--accent: 25 95% 53% (turuncu - bağlantı enerjisi)`
- `--background: 0 0% 98% (açık krem-beyaz)`
- `--foreground: 222 47% 11% (koyu lacivert metin)`
- `--muted: 210 40% 96% (açık gri arka planlar)`
- `--card: 0 0% 100% (beyaz kartlar)`
- `--destructive: 0 84% 60% (kırmızı uyarılar)`

**Typography:**
- Headings: Plus Jakarta Sans (600, 700)
- Body: Work Sans (400, 500, 600)

**Style:** Modern profesyonel, temiz kartlar, cömert spacing, kurumsal ama sıcak

## Features

### Phase 1 (MVP - this session)
- Email + 8-digit membership number authentication
- User profiles (graduation year, department, profession, city)
- Alumni directory & search
- Real-time messaging (one-to-one)
- Gamification basics (activity points, badges)
- Turkish + English language toggle
- Admin roles & permissions
- Fonzip integration (signup form, events embed)

### Phase 2 (future)
- Event management (RSVP, calendar)
- Job postings & referrals
- Mentorship matching
- Groups & forums
- Advanced admin dashboard

### Phase 3 (future)
- WhatsApp community embed
- Recommendation algorithm
- Advanced gamification
